Programme qui gère une application web afin d'utiliser le piano depuis notre téléphone/tablette/ordinateur, sans devoir passer par VNC Viewer.

TOUS les fichiers ont une utilité, ou presque ; attention à ne pas en supprimer. 
De même, il est déconseillé de renommer ou déplacer des dossiers, quels qu'ils soient, à moins d'avoir acquis une compréhension profonde de la dynamique de l'application.

Cette application est peu robuste, et il est possible d'y ajouter des fichiers qui font crasher le programme.
Le site est "protégé" uniquement par le fait qu'il n'est déployé qu'en réseau local : il faut donc connaître le code du réseau afin d'y accéder.
Cependant, tous les PCs de l'établissement y sont liés ; c'est pourquoi une certaine prudence est de mise. Concrètement, les risques sont les mêmes
qu'avec l'ancien système (VNC Viewer), mais l'accès est plus aisé et requiert moins de connaissances.
